# Критерий 6: ЧЕРЕЗ СКОЛЬКО ПЕРВЫЕ ДЕНЬГИ

## Что измеряешь

Сколько времени проходит от начала работы над похожим продуктом до первой оплаты от клиента. По чек-листу цель — 1-2 месяца. Главное — реальные истории, не «теоретические сроки».

## Источники кейсов

Используешь те же издания что и launch-cost, но ищешь специфически «первая продажа» / «первый клиент» истории:

РФ-источники:
- "vc.ru" (раздел firstmilestone)
- "rb.ru"
- "journal.tinkoff.ru"
- "habr.com" — «Я пиарюсь», «Стартапы»
- "incrussia.ru"
- "ko.ru", "the-village.ru", "thebell.io", "republic.ru"

Дополнительно:
- Telegram-каналы основателей — часто пишут «вчера получил первого клиента»
- YouTube подкасты — «Бизнес-секреты», «Запуск завтра», «Подкаст Тинькофф»
- Twitter/X основателей — короткие посты про первые продажи

Глобальные (для проверки если ниша новая в РФ):
- "indiehackers.com" — каталог стартапов с метриками time-to-first-dollar
- "techcrunch.com" — для крупных кейсов

## Что извлекать из каждого кейса

Минимум 5 кейсов с такими полями:

| Поле | Что зафиксировать |
|------|-------------------|
| Кто | Имя + продукт + ссылка |
| Когда стартовал | YYYY-MM |
| Когда первая выручка | YYYY-MM |
| Срок до первой выручки | Дней / недель / месяцев |
| Гео | РФ / СНГ / глобально |
| Сегмент | B2C / B2B / B2G |
| Канал первого клиента | Из списка ниже |
| Сколько заплатил первый клиент | Если указано (₽) |
| Был ли pre-launch (предзаказ/waitlist) | Да / Нет / Не указано |
| Сколько клиентов в первый месяц | Если указано |

## Каналы первых клиентов

Классификация:
1. **Личный нетворк** (warm intro — рекомендации знакомых, бывшие коллеги, клиенты из прошлой работы)
2. **Холодные продажи** (cold email/call, обзвон, рассылки в LinkedIn/TG для B2B)
3. **Контент-маркетинг** (TG-канал основателя, VK-сообщество, YouTube, посты на vc.ru/Habr)
4. **Лид-магнит** (бесплатный продукт/чек-лист в обмен на email, потом продажа)
5. **Платная реклама** (Yandex.Direct, VK Реклама, TG-посевы)
6. **Pre-sale / Waitlist** (собрал лист до запуска, продал при анонсе)
7. **ProductHunt / Habr запуск** (одноразовая публикация которая дала всплеск)
8. **Партнёры / реферралы** (другая компания продаёт за %)
9. **Маркетплейс / агрегатор** (Wildberries, Ozon, AppStore, GetCourse каталог)
10. **Конференция / митап** (выступление = клиент)
11. **Сарафан** (первый клиент сам нашёл — редко в начале, но бывает)

Если в источнике канал не назван явно — отмечаешь «канал не указан в источнике».

## B2B vs B2C цикл в нише

Типичные сроки разнятся радикально:
- **B2C** — от часов (импульсная покупка) до 1-3 недель
- **B2B SMB** (малый бизнес) — обычно 1-3 месяца
- **B2B Enterprise** — 6-18 месяцев цикл продаж
- **B2G** — от 3 месяцев (по госконтракту), иногда 1-2 года

В файле — таблица «характер ниши»:

| Сегмент | Типичный срок до первой продажи | Источники кейсов |

Если ниша допускает несколько сегментов — расписываешь по каждому.

## Активы основателя как ускоритель

Из brief.md (поле «Активы для быстрого старта») — сравни с типовыми каналами:

- Уже есть аудитория в TG/VK/YouTube → канал 3 «контент» работает быстро
- Уже есть нетворк ЛПР (лиц принимающих решения) → канал 1 «личный нетворк»
- Уже есть юзеры из предыдущего проекта → канал 8 «партнёры»
- Уже есть подписчики из инфобизнеса → канал 6 «pre-sale»

Если активы совпадают с быстрыми каналами в нише — это +1 к скорости. Если не совпадают (например, основатель только тех, а ниша B2B и нужны cold sales) — +2-3 месяца к сроку.

```markdown

## Сводка по срокам в нише
- Медианный срок до первой выручки: X
- Минимум: Y
- Максимум: Z
- Распределение: сколько кейсов < 2 мес, 2-6 мес, > 6 мес

## Сводка по каналам
- Какие каналы работают быстрее в этой нише
- Какие доминируют (по частоте)

## B2B / B2C цикл
- Таблица по сегментам

## Совместимость с активами основателя
- Из brief.md: какие активы есть
- Какой канал они активируют
- Реалистичный срок до первой продажи с учётом активов

## Сравнение с целью чек-листа (1-2 месяца)
- Типовой срок в нише: попадает / не попадает / на грани
- Если попадает — какой канал и pre-launch обязательны
- Если не попадает — на сколько отстаёт и почему
```

## Стоп-сигналы для синтезатора

- **Медианный срок > 6 месяцев** — **ДОЛГИЙ ЦИКЛ, не подходит чек-листу**
- **Все кейсы B2B Enterprise (3-18 мес)** — **МЕДЛЕННАЯ НИША**
- **Без сильного нетворка/аудитории кейсы вообще не запускаются** — **НИША ЗАКРЫТА ДЛЯ НОВИЧКОВ БЕЗ АКТИВОВ**
- **Есть кейсы < 30 дней** через лёгкие каналы (контент / pre-sale / реферралы) — **СТАРТ ВОЗМОЖЕН В СРОК**
- **Все требуют B2B-продажи с длинным циклом** — **НЕ ПОДХОДИТ для «первых денег за 1-2 мес»**

## Минимум для VERIFIED

- 5 кейсов с конкретными сроками и каналами, ссылки в каждом
- Из них минимум 3 — последние 2 года ({LAST_YEAR} или {YEAR_BEFORE})
- Сводка по каналам выведена
- Соответствие активам из brief.md разобрано
- Сравнение с целью 1-2 месяца сделано
