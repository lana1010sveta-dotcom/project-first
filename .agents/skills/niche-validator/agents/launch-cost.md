# Критерий 5: СКОЛЬКО СТОИТ ЗАПУСК

## Что измеряешь

Сколько реально потратили другие основатели, чтобы запустить похожий продукт в России и довести его до первой выручки. Ограничение из чек-листа: на старте — **до 300 тысяч рублей**.

Твоя задача — собрать реальные кейсы с разбивкой расходов, не «оценки экспертов», а конкретные истории основателей.

## Источники кейсов запуска

РФ-специализированные:
- **"vc.ru"** (особенно раздел firstmilestone и интервью)
- **"rb.ru"** (РусБейс — стартапы)
- **"journal.tinkoff.ru"** — журнал для предпринимателей
- **"habr.com"** — особенно посты в «Я пиарюсь», «Стартапы»
- **"incrussia.ru"** — Inc. Russia
- **"ko.ru"** — «Компания»
- **"the-village.ru"** — городские предприниматели
- **"thebell.io"** — The Bell
- **"republic.ru"** — длинные материалы

Дополнительно:
- Telegram-каналы основателей — часто раскрывают траты честнее
- YouTube каналы: «Бизнес-секреты», «Подкаст Тинькофф», «Запуск завтра», «ПостПринципал»

## Поисковые запросы (Tavily при TAVILY_MODE=on, иначе WebSearch)

- "как я запустил [ниша] бюджет кейс {LAST_YEAR}"
- "сколько потратил на запуск [ниша] кейс"
- "[ниша] стартап с нуля без инвестиций история"
- "первые шаги в [ниша] предприниматель история"

include_domains: см. выше

## Что извлекать из каждого кейса

Минимум 5 кейсов с такой разбивкой:

| Поле | Что брать |
|------|-----------|
| Кто | Имя основателя + название продукта + ссылка |
| Когда запустился | YYYY-MM |
| Стадия на момент материала | MVP / первые продажи / стабильная выручка |
| Сколько потратил до MVP | Конкретная цифра в ₽ |
| Сколько до первой выручки | Конкретная цифра в ₽ |
| Из чего складывались расходы | Список с цифрами |
| Через сколько окупился | Если указано |

Категории расходов:
1. **Технология** (разработка, серверы, лицензии)
2. **Маркетинг** (реклама, контент, лид-магниты)
3. **Команда** (если нанимал — сколько платил)
4. **Юр.оформление** (ИП регистрация, бухгалтерия, юрист)
5. **Прочее** (домен, хостинг, ОФД — оператор фискальных данных для онлайн-касс)

Если основатель в материале не разбивает по категориям — не додумывай, фиксируй общую сумму с пометкой «разбивка не приведена».

## Типовые цифры РФ-старта (бэйслайн для сравнения)

Используешь как ориентир для проверки кейсов на правдоподобие:

**Юр.оформление ИП:**
- Регистрация ИП — 800 ₽ госпошлина (онлайн бесплатно)
- Открытие счёта (Тинькофф/Точка) — 0 ₽
- Бухгалтерия онлайн (Эльба/Контур.Бухгалтерия) — 6-15 тыс ₽/год

**Технология (минимум для веб-сервиса):**
- Домен .ru — 200-500 ₽/год
- Хостинг (REG.ru/Beget/Timeweb) — 200-1500 ₽/мес
- SSL — бесплатно (Let's Encrypt)
- Бэкенд VPS — 500-3000 ₽/мес
- ChatGPT/Claude API — от $20/мес если AI

**Касса (если по закону нужна):**
- Онлайн-касса — 8-20 тыс ₽ покупка
- ОФД — 3 тыс ₽/год

**Маркетинг (минимум):**
- Yandex.Direct — старт от 1000 ₽
- VK Реклама — от 100 ₽/день
- Реклама в TG-каналах — от 1000 ₽ за пост в средних, 10-50 тыс ₽ в крупных
- Контент в TG/VK — 0 ₽ (своё время)

Если в кейсе цифры сильно расходятся с бэйслайном (например, «50 тыс на хостинг в месяц») — помечай как **требует проверки** в «Пробелы».

## CAC (стоимость привлечения клиента) в нише

Если в кейсах упоминается CAC — собираешь отдельно. Если в нише есть отраслевые отчёты по CAC (для B2B SaaS, e-commerce) — используешь.

Источники CAC: rb.ru, vc.ru, data.vc.ru (исследования), datainsight.ru (для e-commerce), smartranking.ru (для SaaS)

## Что можно пропустить

Отдельный раздел «Что не обязательно для старта»:
- Офис (можно работать из дома)
- Юрлицо ООО (на старте ИП достаточно)
- Свой бренд-дизайнер (можно начать с Tilda + готовые шаблоны)
- Платный CRM (на старте — Notion/Google Sheets)
- Платный таск-трекер (на старте — Trello/Notion)
- Сложная аналитика (на старте — Yandex.Metrika)
- Корпоративная почта (на старте — Gmail/Yandex)

## Структура файла

```markdown
## Кейсы запуска (минимум 5)
[Таблица с разбивкой по каждому]

## Типовые статьи расходов в нише
[Что встречается в большинстве кейсов]

## Реалистичный бюджет старта похожего продукта
- Минимум для MVP: X тыс ₽ (на основе кейсов)
- До первой выручки: Y тыс ₽
- Типовой потолок: Z тыс ₽

## CAC в нише
- Данные если есть, иначе «нет публичных данных»

## Что МОЖНО пропустить на старте
[Список из 3-5 пунктов конкретно для этой ниши]

## Сравнение с ограничением чек-листа (300 тыс ₽)
- Минимум попадает в лимит: ДА / НЕТ
- Реалистичный сценарий попадает: ДА / НЕТ
- Если НЕТ — что нужно сократить или отложить
```

## Стоп-сигналы для синтезатора

- **Все найденные кейсы > 1 млн ₽** на запуск — **НИША ТРЕБУЕТ КАПИТАЛА**
- **Типичный CAC > 30 тыс ₽** — **ДОРОГОЙ ТРАФИК**
- **Требуется лицензирование/сертификация** — **БАРЬЕР ВРЕМЕНИ И ДЕНЕГ**
- **Тяжёлая инфраструктура** (GPU-сервера, региональные ДЦ) — **БАРЬЕР ВХОДА ВЫСОКИЙ**
- **Все кейсы попадают в 300 тыс ₽** — **СТАРТ РЕАЛИСТИЧЕН**

## Минимум для VERIFIED

- 5 кейсов с конкретными бюджетами и ссылками
- Из них минимум 3 — за последние 2 года ({LAST_YEAR} или {YEAR_BEFORE})
- Типовые статьи расходов выделены
- CAC найден ИЛИ явно «нет публичных данных»
- Сравнение с лимитом 300 тыс ₽ сделано
