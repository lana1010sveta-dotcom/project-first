# Критерий 2Б: УГЛУБЛЁННЫЙ РАЗБОР КОНКУРЕНТОВ

## Что делаешь

Берёшь shortlist из `competitors-discovery.md` (передан в prompt) и по каждому из 5-7 конкурентов делаешь полный разбор: JTBD, целевая аудитория, суперсила, каналы трафика, рекламные связки, воронка продаж. В конце — ТОП-5 сильных сторон, которые подтверждены и рекомендуются к «заимствованию» в свою нишу.

JTBD (Job-To-Be-Done) — конкретная задача, которую клиент «нанимает» продукт решить.

## Источники для разбора каждого конкурента

### 1. Лендинг конкурента — WebFetch главной страницы
- Hero-копи раскрывает JTBD: «помогаем X сделать Y чтобы Z»
- CTA — раскрывает целевое действие воронки
- Раздел «для кого» — целевая аудитория сегментирована
- Раздел «как работает» — этапы продукта/онбординга
- Раздел «клиенты/кейсы» — кому подходит на практике

### 2. Блог и кейсы конкурента
- Кейсы клиентов = JTBD устами клиента
- Тон/темы блога = контент-стратегия

### 3. Карточки в каталогах
- vc.ru профиль компании
- Crunchbase профиль
- Профили в каталогах стартапов

### 4. Интервью с основателями
- «Как мы запустились» — раскрывает исходную JTBD
- «Что у нас сработало» — каналы трафика

### 5. Вакансии конкурента (hh.ru, LinkedIn)
- Открытые позиции = где скейлят (продажи / маркетинг / тех)
- JD для маркетологов часто содержат конкретные каналы

### 6. Реклама — что они крутят
- Yandex Wordstat по их бренду + связанным запросам
- Поисковая выдача по их бренду (контекст РСЯ/Google Ads)
- VK Реклама, TG-каналы — где видны их посевы

### 7. Личный бренд основателя (для 🏛 СТАРОЖИЛОВ обязательно)
- Telegram канал founder'а — есть ли, размер, частота
- Подписчики в инстаграм/X
- Выступления на конференциях

## Структура разбора каждого конкурента

```markdown
## [Название конкурента] — [КОГОРТА: 🏛 старожил / 🚀 новичок]

### Базовое
- Основан: YYYY-MM (источник)
- Запущен публично: YYYY-MM (источник)
- Гео: ...
- Основной продукт одним предложением: ...

### JTBD анализ
**Одна главная потребность клиента:** [через JTBD-шаблон — «когда я [ситуация], я хочу [мотивация], чтобы [результат]»]
**Источник формулировки:** [где это вытащено — hero лендинга, кейс клиента, интервью]
**Цитата:** "..."

### Целевая аудитория
- Сегмент: [B2C/B2B/B2G + детализация]
- Конкретный персонаж: [роль в компании / тип физлица]
- Признаки квалификации: [из их же контента]

### Суперсила (что делают ЛУЧШЕ остальных)
[1-2 предложения. Должна быть подтверждена цитатой из независимого источника — отзыв клиента, обзор в прессе, награда. Не маркетинг самого конкурента.]
**Подтверждение:** [URL, цитата]

### Каналы трафика
По убыванию значимости (что находим в публичных источниках):
1. [канал] — [подтверждение: вакансия маркетолога, кейс, выступление основателя]
2. ...

### Рекламные связки (если видно)
- Что промотируют (лид-магнит / триал / демо / прямой оффер)
- На какой посадочной приземляют
- Креативы — если попадались в поисковой выдаче / TG / VK

### Воронка продаж (поэтапно)
Этап 1: [как первое касание]
Этап 2: [что делают после касания — рега / демо / звонок]
Этап 3: [квалификация]
Этап 4: [оффер]
Этап 5: [клоуз / онбординг]

Каждый этап подтверждается из их же материалов (лендинг, лид-магнит, демо-режим, тарифы).

### Личный бренд founder'а (если 🏛 СТАРОЖИЛ)
- Имя founder'а
- Telegram канал / YouTube / Instagram: ссылки + размер аудитории
- Темы контента
- Если личного бренда нет — отметить явно
```

## ТОП-5 СИЛЬНЫХ СТОРОН (раздел в конце файла)

Раздел «Что забираем себе»:

| # | Сильная сторона | У кого подсмотрел | Подтверждение | Применимость для нашего проекта |
|---|-----------------|-------------------|---------------|---------------------------------|

Только то, что:
1. Реально подтверждено цитатой/кейсом (не «они там что-то делают»)
2. Применимо к стадии и ресурсам из brief.md
3. Не требует бюджета > 300к ₽ для воспроизведения

## Стоп-сигналы для синтезатора

- У всех конкурентов одна и та же JTBD-формулировка — **НИША ЗАСТЫЛА В ОДНОМ ПОЗИЦИОНИРОВАНИИ**, нужен новый угол
- Никто из shortlist не растёт по подтверждённым метрикам — **НИША ВЫГОРАЕТ**
- Воронки у всех одинаковые — **ШАБЛОН ПОНЯТНЫЙ**, копировать легко
- Воронки сильно разные — **НЕТ КАНОНА**, нужно искать своё

## Минимум для VERIFIED

- Полный разбор по 5 конкурентам из shortlist
- На каждого минимум 5 фактов с источниками (JTBD, целевая аудитория, суперсила, ≥1 канал трафика, воронка)
- ТОП-5 сильных сторон, каждая с источником
