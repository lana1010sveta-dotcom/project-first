# Критерий 4: КОНСОЛИДАЦИЯ РЫНКА

## Что измеряешь

Насколько рынок захвачен немногими игроками. Главные метрики:
- **CR3** (доля топ-3 игроков) — суммарная доля трёх крупнейших по выручке
- **HHI** (индекс Херфиндаля-Хиршмана) — мера концентрации, от 0 (идеальная конкуренция) до 10000 (монополия)
- M&A-активность (слияния и поглощения)
- Концентрация финансирования

Цель — понять, остаётся ли место для нового игрока.

## Жёсткое правило источников

«Доля рынка» — это не «лидер по узнаваемости». Это **конкретная цифра выручки** игрока на знаменатель **общего объёма рынка**.

Источники долей:
- ✅ Отраслевые рейтинги TAdviser / Smart Ranking / Forbes — где указаны цифры выручки + методология
- ✅ Финансовая отчётность (РСБУ, МСФО) через e-disclosure.ru
- ✅ Данные ФНС / СБИС / Контур.Фокус для конкретных юрлиц
- ✅ Crunchbase для глобальных компаний

Запрещено брать «доли рынка» из:
- ❌ Рекламных материалов самого игрока
- ❌ Статей «лидер рынка» без указания цифры
- ❌ Опросов аудитории «кого вы знаете»

Если конкретных долей нет — переходи к **прокси-метрикам** (см. ниже).

## Пять метрик — собирай по каждой

### 4.1 CR3 / CR5 (концентрация выручки)
- Выручка топ-3 и топ-5 за {LAST_YEAR}
- Общий объём рынка (из market-size.md если уже готов, или из своих находок)
- CR3 = (выручка топ-3 / общий объём) × 100%
- CR5 аналогично

### 4.2 HHI (Herfindahl-Hirschman Index)
Формула: HHI = сумма (доля_i × 100)² для всех игроков с известной долей

Шкала:
- HHI < 1500 — фрагментированный рынок
- 1500-2500 — умеренная концентрация
- 2500-6000 — высокая концентрация (барьер для входа)
- > 6000 — олигополия/монополия

Если доступны только топ-3 — рассчитай HHI floor (минимальная возможная оценка, остальные считаются 0).

### 4.3 M&A-активность
**Источники:** vc.ru, rb.ru, kommersant.ru, forbes.ru, crunchbase.com, mergermarket.com, incrussia.ru

- M&A за последние 24 месяца в нише
- Кто покупатель, кто цель, сумма (если раскрыта)
- Тренд: больше или меньше сделок YoY

### 4.4 Концентрация финансирования
**Источники:** dsight.ru, crunchbase.com, vc.ru, rb.ru
- Топ-5 раундов в нише за {LAST_YEAR}
- Кто получил больше всех денег
- Доля топ-3 получателей в общем объёме инвестиций

### 4.5 Концентрация веб-трафика (прокси)
**Источники:** similarweb.com, semrush.com, liveinternet.ru, radar.yandex.ru
- Доля трафика топ-3 сайтов в категории
- Сравнение с топ-10
- ⚠️ Прокси, не финансовая доля — отмечать явно

## Когда конкретных долей нет

Часть рынков не имеет публичных рейтингов (закрытые B2B-ниши, новые ниши, fragmented рынки услуг). Тогда используешь **3 прокси-метрики одновременно**:

1. Концентрация финансирования (4.4)
2. Концентрация веб-трафика (4.5)
3. Концентрация упоминаний в прессе (по TGStat / медиа-поиск)

Если **все три прокси** показывают концентрацию у одних и тех же 3 игроков → можно делать осторожный вывод о консолидации с пометкой «оценка по прокси».

Если прокси расходятся (по финансированию доминируют одни, по трафику другие) → рынок ФРАГМЕНТИРОВАННЫЙ.

## Стоп-сигналы для синтезатора

- **CR3 > 80% И HHI > 2500** — **РЫНОК КОНСОЛИДИРОВАН, барьер для входа высокий**
- **1 игрок > 60%** — **ПОЧТИ МОНОПОЛИЯ**
- **CR3 < 40% И HHI < 1500** — **ФРАГМЕНТИРОВАН, есть место для входа**
- **3+ крупных M&A за {LAST_YEAR}** — **КОНСОЛИДАЦИЯ В МОМЕНТЕ**
- **Все деньги уходят 1-2 игрокам** — **ПОБЕДИТЕЛЬ ВЫБРАН ВЕНЧУРОМ**
- **Прокси расходятся** — **РЫНОК ФРАГМЕНТИРОВАН по разным осям**

## Минимум для VERIFIED

- CR3 или HHI рассчитан (хотя бы один) — минимум 5 фактов выручки/долей с источниками
- M&A раздел: минимум 3 сделки за 24 мес ИЛИ явное «сделок не было — проверено»
- Концентрация финансирования: топ-5 раундов с суммами
- При отсутствии финансовых данных — минимум 3 прокси с одинаковым выводом
