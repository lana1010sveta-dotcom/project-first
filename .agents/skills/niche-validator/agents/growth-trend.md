# Критерий 3: ДИНАМИКА РЫНКА

## Что измеряешь

Куда движется ниша последние 2-3 года. Используешь **5 независимых количественных индикаторов**. Только сырые цифры по годам, не пересказ нарративов.

## Критично: что НЕ является доказательством роста

❌ «Эксперты прогнозируют рост» — это мнение, не данные
❌ «Рынок активно развивается» — пустая фраза
❌ «По оценкам аналитиков» — без конкретного аналитика и методологии
❌ Одна статья {LAST_YEAR} года говорит «будет 5 млрд к 2030» — это прогноз, а не факт текущего роста

✅ Что является доказательством:
- Сырые числа за конкретные годы из разных независимых индикаторов
- Year-over-year (YoY — изменение год к году) % с конкретными базами расчёта
- Графики с тайм-стэмпами в источниках

## Правила работы с датами

Анализируем динамику **минимум 3 года**:
- {CURRENT_YEAR} — текущий (частичные данные Q1-H1 если есть)
- {LAST_YEAR} — последний полный год (основная точка отсчёта)
- {YEAR_BEFORE} — для расчёта YoY
- {YEAR_BEFORE_LAST} — для проверки тренда 2 года подряд
- Старше {YEAR_BEFORE_LAST} — только как историческая база, не как «сейчас»

## ПЯТЬ ИНДИКАТОРОВ (каждый обязателен)

### 1. Интерес аудитории — разбит на 4 подкатегории

#### 1.1 Поисковые системы
- **Yandex Wordstat**: https://wordstat.yandex.ru/?words=[query] → WebFetch → показы в месяц, динамика
- **Google Trends RU**: https://trends.google.com/trends/explore?geo=RU&q=[query]&date=today%205-y → индекс 0-100 по годам
- **Mail.ru поиск**: https://go.mail.ru/search?q=[query] → косвенный сигнал

#### 1.2 Социальные платформы и комьюнити
- **TGStat**: https://tgstat.ru/search?q=[query] — главное для РФ-инфополя: тематические каналы, динамика упоминаний
- **Habr поиск**: https://habr.com/ru/search/?q=[query] — для tech/B2B
- **VC.ru поиск**: https://vc.ru/search?query=[query] — активность обсуждений
- **VK сообщества**: https://vk.com/search?c[section]=communities&c[q]=[query]
- **Pikabu**: https://pikabu.ru/search?q=[query] — обывательский интерес

#### 1.3 Реальный спрос
- **HH.ru вакансии**: https://hh.ru/search/vacancy?text=[query]&area=113 — B2B-спрос
- **HH.ru резюме**: https://hh.ru/search/resume?text=[query]&area=113 — предложение труда
- **Avito**: https://www.avito.ru/all?q=[query] — для услуг/товаров
- **Госзакупки**: https://zakupki.gov.ru/epz/order/extendedsearch/results.html?searchString=[query] — для B2B/B2G
- **WordStat-кросс**: «купить [ниша]», «цена [ниша]», «заказать [ниша]» — коммерческое намерение

#### 1.4 Содержательное присутствие (контент)
- **YouTube поиск** — новые видео за {LAST_YEAR}
- **Wikipedia pageviews**: https://pageviews.wmcloud.org → ru.wikipedia.org
- **GitHub поиск** для tech-ниш: https://github.com/search?q=[query]
- **ProductHunt** для digital-продуктов

### 2. Вакансии — hh.ru
**Источник:** https://hh.ru/search/vacancy?text=[query]&area=113
- WebFetch на страницу с фильтром по РФ
- Текущее число вакансий
- Сравнение через web.archive.org с {YEAR_BEFORE} и {LAST_YEAR}
- Топ-10 компаний, которые нанимают

### 3. Инвестиционная активность
**Источники:** dsight.ru, vc.ru, crunchbase.com, rb.ru
- Количество раундов: {YEAR_BEFORE}, {LAST_YEAR}, {CURRENT_YEAR}
- Суммарный объём финансирования по годам
- **Раздельно** глобальные и РФ цифры — могут сильно расходиться
- M&A сделки в нише

⚠️ Для РФ-ниш: венчур в РФ сжался с 2022. «Нет раундов» в РФ ≠ «ниша не растёт». Дополни:
- Гранты ФРИИ / Сколково / Иннопром по этой нише
- Господдержка (Минпромторг, Минцифры программы)

### 4. Медиа-присутствие
**Источники:** vc.ru, rb.ru, rbc.ru, forbes.ru, kommersant.ru, vedomosti.ru, tass.ru, cnews.ru, habr.com, thebell.io, incrussia.ru
- Количество публикаций за {YEAR_BEFORE}, {LAST_YEAR}, {CURRENT_YEAR}
- Топ-5 самых заметных публикаций каждого года
- Тренд: растёт ли частота

### 5. Экосистема (специфично под digital-ниши)
Для AI/SaaS/no-code:
- **GitHub**: количество репозиториев + звёзд по теме (через github.com/search или API)
- **ProductHunt** — запуски в категории за 12 мес
- **App stores (RuStore / Google Play)** — новые приложения
- **Конференции** — какие отраслевые ивенты появились {LAST_YEAR}-{CURRENT_YEAR}

Для не-digital ниш — N/A с обоснованием.

## Стоп-сигналы для синтезатора

- **3 из 5 индикаторов** показывают спад YoY — **РЫНОК УВЯДАЕТ**
- Все 5 плоские (±10%) — **ЗАСТОЙ**
- Поиск + медиа растут >50%, но вакансии и инвестиции плоские — **ВОЗМОЖНЫЙ ХАЙП БЕЗ РЕАЛЬНОГО СПРОСА**
- Все 5 растут стабильно 2 года подряд — **РЕАЛЬНЫЙ РОСТ**
- Глобально рост, в РФ плоско — **ТРЕНД НЕ ПРИШЁЛ В РФ ПОКА**

## Минимум для VERIFIED

- По каждому из 5 индикаторов: минимум 3 точки данных ({YEAR_BEFORE}, {LAST_YEAR}, {CURRENT_YEAR}) с источниками
- ИЛИ явно отмечено «N/A для этой ниши» с обоснованием
- Сводная таблица заполнена
- Раздел «Несостыковки» заполнен (или явно «противоречий нет»)
- Итого минимум **12-15 фактов** в файле
