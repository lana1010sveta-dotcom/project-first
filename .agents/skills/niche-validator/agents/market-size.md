# Критерий 1: ОБЪЁМ РЫНКА

## Что измеряешь

Объём рынка ниши в рублях для выбранного гео из brief.md. Только публично подтверждённые цифры. Не TAM/SAM/SOM расчёты «от себя» — только то, что уже посчитано в публичных отчётах, аналитике, финансовых документах.

## Правила работы с датами (строго)

Иерархия свежести (используй placeholder из «Контекст исполнения»):

1. **Приоритет 1** — данные {CURRENT_YEAR} (Q1, H1 если доступны). Реалистично доступны март-май для Q1 текущего года.
2. **Приоритет 2** — данные за полный {LAST_YEAR}. Обычно публикуются в феврале-апреле {CURRENT_YEAR}.
3. **Приоритет 3** — данные {YEAR_BEFORE} как fallback.
4. Данные {YEAR_BEFORE_LAST} и старше — НЕ используй как «текущие», только как историческое сравнение в динамике.

В каждом факте обязательно явно укажи год данных: «в {LAST_YEAR} году» — не «сейчас», не «в последнем году».

## Источник цифр выручки/оборота

«Оборот», «выручка» компаний — только из:
- Финансовая отчётность по РСБУ/МСФО
- Данные ФНС (через egrul.nalog.ru, СБИС, Контур.Фокус)
- Официальные пресс-релизы с цитированием отчётности
- Отраслевые рейтинги от TAdviser/Smart Ranking (они опираются на отчётность)

НЕ используй:
- «Эксперт оценил выручку X в Y» — без указания методологии
- «По оценкам отрасли» — без конкретного аналитика и метода
- Журналистские «около» / «порядка»

## Поисковые запросы (Tavily при TAVILY_MODE=on, иначе WebSearch) (адаптируй под бриф)

- "[ниша] рынок Россия {CURRENT_YEAR} объём"
- "[ниша] объём рынка {LAST_YEAR} факт"
- "[niche] market size Russia {LAST_YEAR} {CURRENT_YEAR}"
- "[ниша] выручка отрасли {LAST_YEAR} финансовая отчётность"
- "[ниша] TAdviser отчёт {LAST_YEAR} {CURRENT_YEAR}"
- "[ниша] Smart Ranking рейтинг {LAST_YEAR}"

## Что должно быть в файле

1. **Объём рынка за последний доступный год** (приоритет {CURRENT_YEAR} → {LAST_YEAR}) — минимум **3 независимых источника**
2. **Объём рынка 2 года назад** для расчёта динамики
3. **Сегментация внутри ниши** если есть разбивка в отчётах
4. **Прогноз CAGR** (средний годовой рост в %) только если опубликован в отчёте со ссылкой на чей это прогноз
5. **Валюта** — если источник в USD, конвертируй с явной указанной датой и курсом ЦБ РФ на эту дату

## Кросс-проверка обязательна

Если все 3+ источника называют один и тот же объём — это повтор друг друга (один первоисточник, все цитируют). Проверь это явно: найди первоисточник, в файле упомяни «3 источника цитируют [первоисточник]» вместо претензии на 3 независимых подтверждения.

## Стоп-сигналы для синтезатора (передаются как часть финального файла)

- Нет ни одной публичной оценки объёма — **НИША НЕДОИЗУЧЕНА**
- Все оценки **< 5 млрд ₽** — **РЫНОК СЛИШКОМ МАЛ**
- Расхождение источников **> 5x** — **ВОПРОС ВЕРИФИКАЦИИ**
- Все источники старше {YEAR_BEFORE} — **ДАННЫЕ УСТАРЕЛИ**

## Минимум для VERIFIED-статуса

- 5 фактов с конкретными цифрами рынка
- Из них минимум 3 за {LAST_YEAR}-{CURRENT_YEAR}
- Из них минимум 2 от источников высшего качества (TAdviser / Smart Ranking / Росстат / IDC)
- Если конкретное юрлицо упомянуто с выручкой — выручка подтверждена из ФНС/СБИС/Контур.Фокус
