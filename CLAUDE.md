# Правила работы с проектом

## Структура файлов

Каждая страница — один самодостаточный HTML-файл. CSS и JS пишутся внутри файла (`<style>` и `<script>`). Внешние библиотеки, фреймворки и CDN не используются.

## Дизайн

- **Фон** — тёмный (близко к чёрному)
- **Текст** — светлый
- **Акцентный цвет** — оранжевый
- Стиль выдержанный, без лишних украшений

## Адаптивность

Все страницы должны корректно отображаться на мобильных устройствах. Обязательно использовать `<meta name="viewport">`, гибкие сетки (`flex`, `grid`) и медиазапросы.

## Стиль текста

Простой и дружеский язык. Без канцелярита, сложных оборотов и официального тона. Писать так, как говорят живые люди.

## Перед созданием нового файла

Сначала рассказать, что планируется сделать: какие секции будут на странице, какой макет, какой контент. Только после подтверждения — приступать к написанию кода.
