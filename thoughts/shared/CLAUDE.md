## Дизайн-система (не менять без необходимости)

Дизайн «Мягкая аврора» — светлая тема с aurora-градиентом.

```css
--a1: #7C3AED   /* фиолетовый */
--a2: #0EA5E9   /* голубой */
--grad: linear-gradient(135deg, #7C3AED 0%, #0EA5E9 100%)
--bg:  #F7F9FC  /* основной фон */
--bgd: #0C1829  /* тёмная секция (портфолио) */
--tx:  #0F1729  /* основной текст */
```

Шрифты: **DM Serif Display** (заголовки, курсив) + **DM Sans** (текст).

Фавикон — inline SVG data URI в `<head>`, буква «S» на градиентном фоне.

## Портфолио — как добавить проект

Найти блок `<!-- PORTFOLIO -->`, добавить карточку в `<div id="pfg">`:

```html
<div class="pc rv" data-t="site">
  <div class="pp" style="background-image:url('img/имя.jpg');background-size:auto 165%;background-position:center 18%"></div>
  <div class="p-tag"><span class="tag t-site">Сайт</span></div>
  <div class="pov">
    <h3 class="p-title">Название проекта</h3>
    <p class="p-meta">Ниша · Описание</p>
    <p class="p-res">✓ Результат</p>
  </div>
</div>
```

Классы карточек: `pc` (обычная), `pc feat` (2×2), `pc wide` (2×1)  
Категории (`data-t`): `site` `landing` `app` `ai`  
Теги (класс): `t-site` `t-land` `t-app` `t-ai`  
Скриншоты класть в `img/`. Параметры фона `background-size:auto 165%; background-position:center 18%` обрезают браузерный хром сверху и тулбар снизу.
