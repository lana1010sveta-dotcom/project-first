# Проект: Сайт-портфолио Светланы

Личный сайт-портфолио веб-разработчика. Одностраничный лендинг + 2 юридические страницы. Хостинг — Netlify.

## Структура файлов

```
thoughts/shared/
├── index.html          ← главная страница (HTML + CSS + JS в одном файле)
├── privacy.html        ← Политика конфиденциальности (ФЗ-152) — ещё не создана
├── terms.html          ← Пользовательское соглашение — ещё не создана
├── brief.md            ← техническая спецификация (справочник, не менять)
├── research.md         ← исследование рынка (справочник, не менять)
└── specs/              ← спецификации проекта (справочник, не менять)
```

## Как редактировать контент

Весь контент — прямо в `index.html`. Файл разделён на понятные секции с HTML-комментариями:

- `<!-- HERO -->` — имя, должность, кнопки связи
- `<!-- ABOUT -->` — текст «О себе», фото
- `<!-- PORTFOLIO -->` — карточки работ
- `<!-- SERVICES -->` — услуги и цены
- `<!-- REVIEWS -->` — отзывы клиентов
- `<!-- CONTACT -->` — форма и контакты
- `<!-- FOOTER -->` — реквизиты, копирайт

## Дизайн-система (не менять без необходимости)

Дизайн «Мягкая аврора» — светлая тема с aurora-градиентом.

```css
--a1: #7C3AED   /* фиолетовый */
--a2: #0EA5E9   /* голубой */
--grad: linear-gradient(135deg, #7C3AED 0%, #0EA5E9 100%)
--bg:  #F7F9FC  /* основной фон */
--bgd: #0C1829  /* тёмная секция (портфолио) */
--tx:  #0F1729  /* основной текст */
```

Шрифты: **Onest** (заголовки) + **Manrope** (текст) — оба поддерживают кириллицу.

## Как добавить работу в портфолио

Найти блок `<!-- PORTFOLIO -->` и добавить карточку по образцу:

```html
<div class="pf-item" data-cat="category">
  <div class="pf-card">
    <div class="pf-img">
      <img src="img/work-name.jpg" alt="Название работы" loading="lazy">
    </div>
    <div class="pf-info">
      <span class="pf-tag">Категория</span>
      <h3>Название</h3>
      <p>Короткое описание</p>
      <a href="https://ссылка.на.сайт" class="pf-link" target="_blank">Смотреть →</a>
    </div>
  </div>
</div>
```

Категории (`data-cat`): `site` `landing` `bot` `quiz` `other`

## Как добавить отзыв

Найти блок `<!-- REVIEWS -->`, добавить по образцу:

```html
<div class="rv-card">
  <p class="rv-text">«Текст отзыва»</p>
  <div class="rv-author">
    <strong>Имя Фамилия</strong>
    <span>должность, компания</span>
  </div>
</div>
```

## Форма заявки

Форма подключена к **Netlify Forms** (`data-netlify="true"`). Письма приходят на email, привязанный к аккаунту Netlify. Не трогать атрибуты `data-netlify`, `netlify-honeypot` и скрытое поле `form-name`.

## Контакты — где менять

Все контакты встречаются в двух местах: секция `<!-- CONTACT -->` и `<!-- FOOTER -->`.

Плейсхолдеры для замены:
- `@username` → Telegram-юзернейм
- `wa.me/your_number` → номер WhatsApp в формате `wa.me/79XXXXXXXXX`
- `tel:+your_number` → номер телефона
- `your@email.com` → email

## Юридические страницы

Ещё не созданы. Создать как отдельные файлы `privacy.html` и `terms.html` рядом с `index.html`. Перед созданием — уточнить ФИО, адрес, ИНН (заполнить после регистрации ИП).

## Деплой

Сайт деплоится на **Netlify** автоматически при пуше в ветку `main` на GitHub (`lana1010sveta-dotcom/project-first`).

После пуша проверить: Netlify Dashboard → Deploys → убедиться что deploy прошёл.

## Правила работы

- Редактировать только `index.html`, `privacy.html`, `terms.html`
- Файлы `brief.md`, `research.md`, `specs/` — только для чтения, справочник
- Не менять дизайн-систему (цвета, шрифты) без явного запроса
- После изменений — коммитить и пушить в `main`
- Изображения для портфолио класть в папку `thoughts/shared/img/` (создать при первом добавлении)
