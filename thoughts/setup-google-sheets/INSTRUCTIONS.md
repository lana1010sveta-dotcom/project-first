# Подключение Google-таблицы к заявкам с сайта

Цель: чтобы каждая заявка с сайта, кроме почты и Telegram, **записывалась строкой в Google-таблицу**.

Код сайта (`api/contact.js`) уже умеет это — он шлёт заявку на адрес из переменной
`GOOGLE_SHEET_URL`. Осталось создать таблицу-приёмник и вписать её адрес в Vercel.

Время: ~5–7 минут. Всё кликается мышкой, программировать ничего не нужно.

---

## Шаг 1. Создать таблицу

1. Открой https://sheets.new (создастся новая Google-таблица).
2. Назови её, например, **«Заявки с сайта SvetlanaDev»** (вверху слева).
3. Внизу переименуй вкладку листа в **`Заявки`** (двойной клик по «Лист1» → ввести `Заявки`).
   *(Если не переименуешь — скрипт всё равно запишет в активный лист, но так аккуратнее.)*

## Шаг 2. Открыть редактор скриптов

В меню таблицы: **Расширения → Apps Script** (Extensions → Apps Script).
Откроется редактор кода в новой вкладке.

## Шаг 3. Вставить код

1. Удали весь текст, который там есть по умолчанию (`function myFunction() {}`).
2. Открой файл **`Code.gs`** из этой же папки, скопируй его содержимое целиком и вставь в редактор.
3. Нажми **значок дискеты** (Сохранить), Ctrl+S.

## Шаг 4. Опубликовать как Web App

1. Вверху справа: **Развернуть → Новое развёртывание** (Deploy → New deployment).
2. Нажми шестерёнку рядом с «Тип» → выбери **Веб-приложение** (Web app).
3. Настройки:
   - **Описание:** любое, например `приёмник заявок`.
   - **Запуск от имени:** **Я** (Me / твой аккаунт).
   - **У кого есть доступ:** **Все** (Anyone). ⚠️ Это обязательно — иначе сайт не сможет писать.
4. Нажми **Развернуть** (Deploy).
5. Google попросит **разрешения** → выбери свой аккаунт → «Дополнительно» →
   «Перейти к проекту (небезопасно)» → **Разрешить**. *(Это нормально: ты разрешаешь
   собственному скрипту писать в собственную таблицу.)*
6. Скопируй **URL веб-приложения** — он длинный и заканчивается на **`/exec`**.
   Например: `https://script.google.com/macros/s/AKfyc.../exec`

> 💡 Проверка: открой этот URL в браузере — должна появиться надпись
> «SvetlanaDev — приёмник заявок работает ✅». Значит, приёмник жив.

## Шаг 5. Вписать адрес в Vercel

1. Зайди на https://vercel.com → проект сайта (`my-website`).
2. **Settings → Environment Variables**.
3. Добавь переменную:
   - **Name:** `GOOGLE_SHEET_URL`
   - **Value:** тот самый URL, что заканчивается на `/exec`
   - **Environments:** отметь **Production** (и Preview, если хочешь тест).
4. **Save**.

## Шаг 6. Передеплоить (обязательно!)

Переменные окружения применяются только при новом деплое:
- Vercel → вкладка **Deployments** → у верхнего деплоя меню «…» → **Redeploy** → подтвердить.

## Шаг 7. Проверить

1. Открой сайт, заполни и отправь форму заявки (можно тестовыми данными).
2. Загляни в таблицу — должна появиться новая строка с датой, именем, контактом и т.д.
3. Заодно проверь, что заявка как и раньше пришла в Telegram и на почту.

---

## Если что-то не записалось

- **URL не на `/exec`** — скопировался не тот адрес. Должен быть Web app URL из развёртывания.
- **Доступ не «Все»** — переразверни (Deploy → Manage deployments → ред. → Anyone).
- **Не передеплоил Vercel** после добавления переменной — повтори Шаг 6.
- **Поменял код скрипта позже** — нужно заново «Развернуть» (новая версия развёртывания).

Когда подключишь — скажи мне, я помогу проверить и при желании настрою формат
таблицы (заголовки, авто-ширину, уведомления).
