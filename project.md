# Svetlana Digital Studio — Project Description

## Что это за проект

Это персональное портфолио Светланы — веб-дизайнера и разработчика студии **Digital Studio**. Репозиторий объединяет личную визитную карточку автора и примеры клиентских работ. Проект демонстрирует навыки в создании сайтов под ключ, веб-приложений и ИИ-агентов.

## Для кого

- **Потенциальные клиенты** — те, кто ищет разработчика для создания сайта, лендинга или веб-приложения.
- **Партнёры и коллеги** — для знакомства с уровнем работ и стилем автора.
- **Клиенты «Цветочной лавки»** — покупатели авторских букетов с доставкой, которые ценят красоту и внимание к деталям.

## Существующие страницы

| Файл | Назначение |
|------|-----------|
| `vizitka/vizitka.html` | Личная цифровая визитка Светланы: услуги, соцсети, кнопка для связи в Telegram |

## Планируемые страницы

- **Каталог / портфолио работ** — галерея выполненных проектов с описанием и ссылками
- **Страница контактов** — форма обратной связи или карточка с реквизитами
- **Галерея букетов** — страница с фотографиями букетов для лендинга «Цветочная лавка»

## Стиль дизайна

Обе страницы выполнены в едином премиальном стиле:

- **Цветовая палитра** — тёмный фон (`#06070a`), золотые акценты на визитке, розово-бордовые на лендинге
- **Типографика** — Georgia (serif) для заголовков, системный sans-serif для текста
- **Эффекты** — анимированные ambient-орбы на фоне, мерцающие частицы, плавные scroll-reveal анимации
- **Карточки** — glassmorphism: полупрозрачный фон, тонкая рамка, hover с подъёмом и свечением
- **Адаптивность** — все страницы корректно отображаются на мобильных устройствах
